Github Search

Github Search

Github search is a github username / repository search engine that allows the user to input a github user's username or repository. The search returns a list of all the persons repositories and description of those repositories.

Application Design

The aim was to create a simple github search engine clone.

The user enter a github username pf the person they wish to search for, they are then presented by a list of all the persons repositories and their description .

Finally, the user can click on a specific repository to get more information on it.

Technologies used: Javascript-HTML & CSS-Node-Moment.js-Github API
